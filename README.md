# Sariva AI

Sariva AI is an intelligent interview and assistant platform built using Streamlit and Groq LLM.

## Features

- Resume-aware AI Assistant
- Entry-level AI Interviewer
- Voice-based answers
- Automated evaluation
- Indian neural voice output

## Setup

1. Clone repository
2. Create virtual environment
3. Install dependencies:
   pip install -r requirements.txt
4. Add GROQ_API_KEY in .env
5. Run:
   streamlit run app.py
